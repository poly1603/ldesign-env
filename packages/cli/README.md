# @ldesign/env-cli

> 环境配置管理 CLI 工具

## 安装

```bash
pnpm add -g @ldesign/env-cli
```

## 使用

```bash
# 初始化
ldesign-env init

# 切换环境
ldesign-env use production

# 验证配置
ldesign-env validate

# 启动 Web UI
ldesign-env serve
```
